◆ match()

auto grapher::predicates::match ( grapher::json_t const &  constraint)
inline

Generates a match predicate from constraint. A match predicate returns true if all the fields contained in the matcher object exist in the observed value and hold the same values.

String values will be regex matched if "regex_match" option is set to true.

Examples:

Using regex match:

{
"type": "match",
"regex_match": true,
"matcher":
{
"name": "Source",
"args":
{
"details": "/usr/include/boost/*"
}
}
}

Not using regex match:

{
"type": "match",
"regex": false,
"matcher":
{
"name": "Source",
"args":
{
"details": "/usr/include/boost/preprocessor/logical/bool.hpp"
}
}
}
