Plotter configuration documentation

compare

For each group descriptor, generates a graph comparing all benchmark cases in the set.

Plotter-specific JSON parameters:

  • value_json_pointer (string): pointer to JSON value to measure
  • draw_average (bool): enable average curve drawing
  • draw_points (bool): enable point value drawing

  • group_descriptors (group descriptors): see group_descriptor_t documentation

    Common plot JSON parameters:

  • width (int): Graph width
  • height (int): Graph height
  • legend_title (string): Graph legend title
  • x_label (string): X axis label
  • y_label (string): Y axis label
  • plot_file_extensions (string array): List of extensions for the export

Example config:

{
"draw_average": true,
"draw_points": true,
"group_descriptors": [
{
"name": "All",
"predicates": [
{
"pointer": "/name",
"regex": "*",
"type": "regex"
}
]
}
],
"height": 500,
"legend_title": "Timings",
"plot_file_extensions": [
".svg",
".png"
],
"plotter": "compare",
"value_json_pointer": "/dur",
"width": 1500,
"x_label": "Benchmark size factor",
"y_label": "Time (µs)"
}

compare_by

Generates a benchmark comparison graph for every matching key built from the list of JSON pointers.

JSON config parameters:

  • key_ptrs (string array): pointers to JSON values to use as a key
  • value_ptr (string): pointer to the JSON value to measure
  • draw_average (bool): enable average curve drawing
  • draw_points (bool): enable value point drawing
  • demangle (bool): demangle C++ symbol names

  • filters (array, optional): array of predicates to filter observed events. Useful for large datasets.

    Returns a default configuration for apply_config.

    Common plot JSON parameters:

  • width (int): Graph width
  • height (int): Graph height
  • legend_title (string): Graph legend title
  • x_label (string): X axis label
  • y_label (string): Y axis label
  • plot_file_extensions (string array): List of extensions for the export

Example config:

{
"demangle": true,
"draw_average": true,
"draw_points": true,
"filters": [
{
"pointer": "/name",
"regex": "*",
"type": "regex"
}
],
"height": 500,
"key_ptrs": [
"/name",
"/args/detail"
],
"legend_title": "Timings",
"plot_file_extensions": [
".svg",
".png"
],
"plotter": "compare_by",
"value_ptr": "/dur",
"width": 1500,
"x_label": "Benchmark size factor",
"y_label": "Time (µs)"
}

debug

Debug plotter. Outputs various statistics on benchmark categories to debug category building or traversal issues.

Example config:

{
"plotter": "debug"
}

stack
